تعزيز الأمن السيبراني في الدفاع

وَكَّلَتْ وزارة الدفاع الأميركية (البنتاغون) قَاعِدَةَ نَمُوذَج نُضُوجِ الأَمَانِ السَّيبراني (CMMC)، والتي تهدف إلى تعزيز ممارسات الأمان السيبراني في قطاع الدفاع. هذه الخطوة الهامة ستُمهِّد الطريق لتنفيذ معايير أمان سيبراني إلزامية عبر جميع مقاولي الدفاع. الإطار الجديد سيعزز وضع الأمان لسلسلة إمدادات الجيش الأميركي، مُعالجةً الثغرات وضمان حماية المعلومات الحساسة بشكل أفضل.
