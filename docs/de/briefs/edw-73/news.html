Eine solche Maßnahme ist die Implementierung einer Zero-Trust-Architektur, einem Sicherheitsmodell, das eine strenge Identitätsüberprüfung für jeden Benutzer und jedes Gerät verlangt, das versucht, auf Ressourcen in einem Netzwerk zuzugreifen.
