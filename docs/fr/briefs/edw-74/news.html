High-Severity Command Injection Flaw in Vanna.AI Library Exposes Databases to Remote Code Execution Attacks; Users Advised to Run Library in Segregated Environment for Risk Mitigation.

Une faille d'injection de commandes de haute gravité dans la bibliothèque Vanna.AI, suivie sous le numéro CVE-deux mille vingt-quatre-cinq mille cinq cent soixante-cinq, expose les bases de données à des attaques d'exécution de code à distance (RCE). Cette vulnérabilité permet aux attaquants d'exécuter des commandes arbitraires via la fonction "demander", pouvant potentiellement entraîner des failles de sécurité importantes. Vanna a émis un guide de durcissement conseillant aux utilisateurs d'exécuter la bibliothèque dans un environnement cloisonné pour atténuer les risques. Ceci met en évidence les dangers de l'utilisation répandue de l'intelligence artificielle générative sans gouvernance et mesures de sécurité appropriées.
