Cybersecurity

High-risk security breach in Vanna.AI library exposes databases to remote code execution attacks, highlighting AI security hazards.

Una grave lacuna di iniezione immediata di elevata gravità nella libreria Vanna.AI, tracciata come CVE-duemilaventicinque-cinquecinquesei, espone le basi di dati ad attacchi di esecuzione di codice remoto (RCE). Tale vulnerabilità consente agli aggressori di eseguire comandi arbitrari attraverso la funzione "ask", potenzialmente portando a significative violazioni di sicurezza. Vanna ha emesso una guida di rinforzo che consiglia agli utenti di eseguire la libreria in un ambiente sandboxed per mitigare i rischi. Ciò evidenzia i pericoli dell'uso diffuso di intelligenza artificiale generativa senza adeguate misure di governance e sicurezza.

Microsoft Unveils New Cybersecurity Threat 'Skeleton Key AI Jailbreak', Capable of Bypassing Authentication Systems

Microsoft ha rivelato una nuova minaccia per la sicurezza informatica, 'Skeleton Key AI Jailbreak,' in grado di eludere i sistemi di autenticazione. La divulgazione mira a sensibilizzare e stimolare gli sforzi di mitigazione tra le organizzazioni in tutto il mondo.

Researchers identify new supply chain attack 'Polyfill', compromising third-party website libraries

I ricercatori di sicurezza hanno identificato un nuovo attacco alla catena di fornitura chiamato "Polyfill".
