Akash Nigam

Akash Nigam

LinkedIn Profile

アカシュ・ニガムは、1992年3月1日にカリフォルニア州ロスアルトスで生まれたアメリカの起業家兼投資家です。彼はミシガン大学に通い、2010年から2014年までコンピュータサイエンスと経済学を学びました。ニガムはエバン・ローゼンバウムと共にソーシャルメディアプラットフォームBlendを共同設立し、その後2017年にGeniesに転換しました。彼の両親は、300万ドルを調達するか、学校に留まるかの選択を彼に与えました。ニガムは中退することを選び、その資金を調達したことで、デジタルプラットフォーム向けのパーソナライズされた3DアバターのリーダーであるGeniesの成功を導きました。ニガムは2019年にフォーブスの30歳未満の30人に選ばれ、テクノロジーとデジタルイノベーションの分野で引き続き著名な人物となっています。
