Dave Burrill

Dave Burrill

LinkedIn Profile

シニアテクノロジーエグゼクティブ（社長、COO、CIO）、起業家、著者、講演者、取締役およびアドバイザー。共同設立または初期段階から、資金調達ラウンドを経て、急速な収益成長（オーガニックおよび買収）を遂げ、IPOまたは上場企業への売却を果たした三つの成功したテクノロジー企業の経営チームメンバー。ビジネスとITの分野を橋渡しする専門知識を持つ信頼されるアドバイザーおよびC-Suiteの同僚。スタートアップからフォーチュン50に至るクライアントのために成功裏にプロジェクトを完了した、多分野・多地域のチームを率いるテクノロジープロジェクト管理の経験が30年以上。

元PWCコンサルタントおよび公認会計士（CPA）、アメリカ公認会計士協会（AICPA）、プロジェクトマネジメント協会（PMI）、スクラムアライアンスのメンバーであり、認定プロジェクトマネジメントプロフェッショナル（PMP）およびスクラムマスター（CSM）。SaaS、高度な分析および機械学習における業績で国際的な賞を受賞。PMI、IEEE、IIBA、SIMなどの組織で、エンタープライズITプロジェクトの高い失敗率、その失敗の75%に関わるコミュニケーションの問題、そのコミュニケーション問題の神経学的原因、そして比較的簡単な解決策について頻繁に講演している。

南カリフォルニア大学マシャルビジネススクールで組織とシステムデザインに関する博士号を追求し、MBAを取得。カリフォルニア大学アーバイン校で経済学および心理学の学士号を取得している。
