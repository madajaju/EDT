Michael Lenox

Michael Lenox

LinkedIn Profile

マイケル・レノックスは、受賞歴のある教授、著者、講演者、コンサルタントです。25年以上にわたり、彼はMBA学生や経営者がイノベーションと混乱の中で市場の競争力ダイナミクスを乗り越える手助けをしてきました。脱炭素化からデジタル変革まで、マイクは今日の経営者にとって最も重要なトピックに関する広範な専門知識を持っています。戦略コンサルタントとして、彼はゼネラルモーターズ、ドミニオンエナジー、NASA、NCAAなどの多くの組織が効果的な戦略を策定する手助けをしてきました。オンライン教育者として、彼は数百万の学習者に戦略的思考と推論能力を向上させる手助けをしています。マイクはバージニア大学ダーデンスクールオブビジネスのテイロー・マーフィー経営学教授です。彼はデューク大学とNYUの教員を務め、ハーバード大学、スタンフォード大学、オックスフォード大学の客員教授としても活動してきました。彼はMITでテクノロジーマネジメントと政策の博士号を取得し、UVAでシステム工学の学士号と修士号を取得しています。
