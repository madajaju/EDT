共同創設者兼CEO、MemVerge

チャールズ・ファンは、Big Memory SoftwareのリーダーであるMemVergeの共同創設者兼CEOです。MemVergeは、企業向けにメモリ・アズ・ア・サービスとGPU・アズ・ア・サービスを提供しています。

MemVergeの前は、チャールズはモバイルインターネット企業Cheetah MobileのCTOを務めていました。彼はAI、大データ、クラウドにおける技術戦略を導き、グローバルなエンジニアリングプラットフォームチームを管理していました。

Cheetahの前、チャールズはVMwareでSVP/GMとして活躍し、VMwareのストレージビジネスユニットとビッグデータグループを担当していました。彼は、業界を変革する製品であるVirtual SANを含むチームを率いました。

VMwareの前、チャールズはEMCで働き、EMC中国R&Dセンターの創設者でした。彼はRainfinityの買収を通じてEMCに入社しました。彼はRainfinityの共同創設者兼CTOであり、そのファイル仮想化および高可用性製品の発明者でした。

チャールズはカリフォルニア工科大学で電気工学の博士号および修士号を取得し、クーパー・ユニオンで電気工学の学士号を取得しました。
