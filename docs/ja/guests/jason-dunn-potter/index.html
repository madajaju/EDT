Biography

ジェイソン・ダン＝ポッターはインテル社で米国公共部門のソリューションアーキテクトを務めています。彼は政府機関や産業パートナーと直接協力し、信頼されるリーダーシップと率直なフィードバックを提供することで複雑な問題を解決しています。

ジェイソンは、米国公共部門、国防総省、国家安全保障機関でほぼ30年間働いており、最適化の推進、革新的な変化の導入、技術専門家の統一チームのリーダーシップを行ってきました。これにより、政府と産業の両方に直接利益をもたらす多様な技術において深い知識を持っています。彼は、移動型プラットフォーム（空、陸、海、宇宙）及び固定サイトアーキテクチャに関して広範な経験を有しています。彼は内部チーム、エコシステムパートナー、政府機関との持続的な関係を構築し、育んでいます。

ジェイソンは、AFCEA、AUSA、USAWOA（ロンスターチャプターの現副会長）、SCRA、ウォリアーズ・エトスのシニアメンターなど、多くの専門団体の誇り高きメンバーです。彼は産業の業界誌や国防総省のドクトリン向けに複数の記事を発表しています。

インテルに入社する前、ジェイソンは26年以上にわたりアクティブアーミーの兵士として通信士（信号軍）に従事し、伍長専任官（CW5）として退役しました。彼の以前の任務にはホワイトハウス通信局（WHCA）のCTOとしての役割、15年以上にわたる複数の戦闘部隊（戦術的任務）、ほぼ5年間のネットコム組織（戦略的任務）、および様々な管理職が含まれます。彼は、イラク、アフガニスタン、その他10か国以上への多数の展開を含む、世界中でさまざまな技術リーダーシップの役割を果たしてきました。彼は、ネットワーク、システム、サーバー、サイバーセキュリティ、管理を含むITのさまざまな側面において、ほぼ1ダースの業界認定を受けています。ジェイソンは、情報技術の修士号や10年以上にわたるCISSP資格を保持しています。
