Rob Atkinson

Rob Atkinson

LinkedIn Profile

情報技術とイノベーション財団（ITIF）の創設者兼会長であるロバート・D・アトキンソンは、科学技術政策における世界最高のシンクタンクとして認識されており、技術革新と公共政策の交差点における一連の重要な問題に関する議論を形成し、議題を設定することに成功している政策アナリストやフェローの豊かなチームを率いています。

彼は国際的に認められた学者であり、多くの著書を持つ著者であり、The New Republicによって「イノベーションに関する最も重要な三人の思想家」の一人に挙げられ、Washingtonian Magazineによって「テックタイタン」と呼ばれ、Government Technology Magazineによって「情報技術の25人のトップの実践者、夢想家、推進者」の一人として評価され、ウォートンビジネススクールから「ウォートン・インフォシス・ビジネス変革賞」を授与されています。

求められるスピーカーであり、世界中の政策立案者にとって貴重なアドバイザーであるアトキンソンの著書には、『テクノロジーの恐れとスケープゴート：プライバシー、雇用、AI、そして今日のイノベーション経済に関する40の神話』（パルグレーブ・マクミラン、2024）、『大きさは美しい：小規模ビジネスの神話を打破する』（MITプレス、2018）、『イノベーション経済学：グローバル優位性の競争』（イェール、2012）、『供給サイドの愚行：なぜ保守経済学は失敗し、リベラル経済学はつまずき、イノベーション経済学が答えなのか』（ローマン・アンド・リトルフィールド、2006）、『アメリカ経済の過去と未来：成長のサイクルを支える長いイノベーションの波』（エドワード・エルガー、2005）があります。彼はまた、税政策から先進製造業、生産性、グローバル競争力に至るまで、技術やイノベーションに関連するトピックに関する数百の論文や報告書を発表し、画期的な研究プロジェクトを実施してきました。彼はアメリカ合衆国議会に30回以上証言しています。

クリントン大統領はアトキンソンを新経済における労働者、コミュニティ、および経済変革に関する委員会に任命し、ブッシュ政権は彼を議会が設立した国家表面輸送インフラ資金調達委員会の委員長に任命しました。オバマ政権は彼を国家イノベーションと競争戦略アドバイザリーボードに任命し、ホワイトハウス科学技術政策局の中国・アメリカイノベーション政策専門家グループの共同議長として、アメリカ合衆国商務省の国家イノベーションおよび起業家精神に関する諮問委員会のメンバーとして、そしてトランプ政権は彼をG7人工知能グローバルパートナーシップのメンバーに任命しました。バイデン政権は彼をアメリカ合衆国国務省の国際通信と情報に関する諮問委員会のメンバーおよびアメリカの輸出入銀行の中国競争に関する評議会のメンバーに任命しました。

アトキンソンはまた、イギリス政府の地域諮問グループのメンバーとして、科学、研究、およびイノベーション大臣に対してどのように政策がより多くの地域でイノベーションを推進できるかについて助言しました。彼は、アメリカ合衆国政府監査局の科学技術評価および分析（STAA）チームに新たな問題に関する助言を行う、さまざまな分野の学際的な科学技術政策専門家の団体であるポラリス評議会のメンバーでもあります。

アトキンソンは特別競争研究プロジェクトのメンバーでもあります。彼は情報時代の国家安全保障に関するマーケル財団タスクフォースで活動し、オレゴン大学の政策研究およびイノベーション学部と州科学技術学院の理事会または諮問委員会に参加しています。さらに、アトキンソンは『電子政府ジャーナル』と『インターネット政策ジャーナル』の編集委員会のメンバーであり、グローバルイノベーションフォーラムのブレイントラストのメンバー、ブルッキングス研究所の非居住シニアフェロー、コロンビア大学テレインフォメーション研究所のフェロー、東京に本拠を置く研究所であるグロコムのフェローでもあります。彼はまた、ジョージタウン大学外交サービス学部の非常勤教授でもあります。

アトキンソンは以前、プログレッシブ政策研究所の副会長を務め、そこで技術と新経済プロジェクトを指揮しました。彼はブロードバンド通信、電子商取引、電子政府、プライバシー、著作権、研究開発税政策、オフショアリング、イノベーション経済学などの問題をカバーした技術およびイノベーション政策に関する多数の研究報告書を執筆しました。

以前、アトキンソンはロードアイランド州経済政策評議会（RIEPC）の初代エグゼクティブディレクターを務めており、そのメンバーには州知事、立法指導者、企業および労働指導者が含まれていました。RIEPCの責任者として、彼は州の包括的な経済開発戦略の立案を担当し、立法府および行政部門と協力して10項目の行動計画の各要素を円滑に実行するために取り組みました。

ロードアイランド州での活動に先立ち、アトキンソンは連邦議会技術評価局のプロジェクトディレクターを務め、情報技術革命がアメリカの都市地域に与える影響を考察した重要な報告書『アメリカのメトロポリスの技術的再形成』を主導しました。

尊敬される政策専門家および解説者として、アトキンソンはアメリカ合衆国上院および下院で数多くの証言を行い、ニュースや公共の問題に関する番組に頻繁に出演しています。これらの出演には、BBC、CNBC、CNN、Fox News、MSNBC、NPR、NBCナイトリーニュースでのインタビューが含まれています。

アトキンソンは、ノースカロライナ大学チャペルヒル校で都市および地域計画の博士号を取得し、そこで名誉あるジョセフ・E・ポグ奨学金を授与されました。また、オレゴン大学で都市および地域計画の修士号を取得し、2014年には優れた卒業生として認定されました。
