Biography

スティーブ・オリンはインテルの連邦CTOおよびシニアプリンシパルエンジニアです。彼は公共部門のソリューションアーキテクチャ、戦略、技術エンゲージメントをリードしており、インテルではサイバーセキュリティプログラム、製品、および戦略を主導する技術リーダーシップのポジションに就いていました。スティーブは以前、サーヴェガのCSO、サンクタムのCTO、ロックスターハンズのCTOおよび共同創設者、シンダータ・テクノロジーズのCTOを務めていました。スティーブは企業セキュリティの著名な専門家であり、頻繁に講演を行っています。彼はInfoWorldのトップ25 CTOに名を連ね、エグゼクティブ・モザイクのトップCTO エグゼクティブ賞を受賞し、2023年のワシントンエグゼクの注目すべき最高技術責任者に選ばれ、NSITC/IDESGセキュリティ委員会の副議長を務め、NISTの国家サイバーセキュリティセンター・オブ・エクセレンス（NCCoE）でゲストリサーチャーを務めました。彼は高度防衛研究センターのフェローであり、INSAサイバー委員会の議長です。
