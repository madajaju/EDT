Richard Broo

Biography

リチャード・ブルーは、True North PMP Consultingの創設者兼代表であり、グローバルな複合材料およびプラスチック産業で40年以上の成功したリーダーシップ経験を持っています。彼は、営業、マーケティング、製品管理、プロジェクト管理、新製品開発および商業化、運営において成功裏にリーダーシップの役割を果たしてきました。彼のキャリアは、個々の貢献者から社長/CEOの役職に至るまでの任務を含んでいます。

True North PMP Consultingを設立する前、リチャードはブラック＆デッカーオートモーティブ、オーウェンズ・コーニングオートモーティブ、ゼネラル・エレクトリック・プラスチックスなどのフォーチュン500企業や、旭化成プラスチックス、ヘキシオンスペシャリティケミカルズ、クアドラントプラスチック複合材料などの中堅企業で働いていました。

彼のコンサルタントとしての専門知識には、戦略的計画、商業的成長、製品管理、プロジェクト管理、新製品開発と商業化、運営、失敗しているビジネスの再建における計画、実施、最適化が含まれます。彼は、効果的なプロジェクト管理を利用してコスト、サイクルタイム、リスク、無駄を削減し、競争の差別化と収益性を高める方法について企業を指導/メンタリングします。

リチャードは、戦略的ビジョン、クロスファンクショナルチームワーク、プロセスの規律、顧客の声を使用して、収益、成長、運営、収益性を最適化できる高い効果的な問題解決者および価値創造者として特徴付けられています。彼は、挑戦的な環境で卓越する方法を知っている実績のあるチームリーダーでありチームプレイヤーです。リチャードが他のコンサルタントと異なる点は、クライアントに理論ではなく、具体的で検証可能な結果を提供し、目的を達成する能力です。

リチャードはウォルシュカレッジで経営学修士号を取得し、西メリーランド大学で文学士号を取得しています。彼は認定プロジェクトマネジメントプロフェッショナル（PMP）です。
