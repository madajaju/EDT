Joel Shertok

Joel Shertok

LinkedIn Profile

実践的な化学工学/材料コンサルタントであるジョエル・シャートク博士の専門は、新製品開発、商業製造、化学プロセスの最適化です。シャートク博士は、複数の技術、化学、さまざまなユニットオペレーションにおける豊富なバックグラウンドを活用し、収益性の高い新製品、樹脂、中間体、プロセスを生み出すプログラムを開発してきました。

シャートク博士は、さまざまな技術にわたって新しい製品を開発してきました。特別な化学物質（ポリマー、重合コーティング）、炭素繊維複合材料、バイオテクノロジーに基づく栄養補助食品、膜ろ過、イオン交換、炭素ベースの分離および調合開発です。

プロセス/製品開発のシニアエグゼクティブとして、シャートク博士は新製品/プロセス戦略の創出に関わる意思決定プロセスに参加してきました。

最近のシャートク博士のコンサルティング活動には以下が含まれます：

  • NYSERDA（ニューヨーク州エネルギー研究開発公社）の起業家インレジデンス - バイオテクノロジーおよび化学合成におけるスタートアップ企業がプロセスを商業化するためのアドバイスを提供。
  • スタートアップのナノテクノロジー企業向けのナノ金属合成プロセスの商業化。
  • 逆浸透、水処理化学物質、公害対策、接着剤、水フィルターなどの分野におけるマーケティング調査の作成。
  • 竹を原料としてバイオエタノールおよび関連バイオ燃料を製造する発酵プロセスの開発/商業化。
  • プロセス最適化のための機械学習の応用。
  • Liバッテリー用の専有電解質を製造する新しいプロセスのスケールアップと商業化。
  • プロセス最適化およびスタートアップ製薬診断ラボの商業ソフトウェア会社の拡大をサポート。
  • スクラバー水からセレンを除去するためのメディアの商業化。

シャートク博士はプリンストン大学（ニュージャージー州プリンストン）で化学工学の博士号を取得し、クーパー・ユニオン（ニューヨーク州ニューヨーク）で化学工学の学士号を取得しています。多数の特許を保有しています。シャートク博士はRITの化学および材料科学の学校で客員研究員として勤務していました。彼はさまざまな精製技術において多くの特許を持っています - ろ過、炭素ブロックなど。
