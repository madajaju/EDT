Biography

20年以上にわたり、私は戦略的意思決定の中心で働いてきました — リーダーが成長、変革、複雑さを明確さと自信を持って乗り越える手助けをしています。

私の経歴は、国際的なCFOの役割、実践的なM&Aの実行、取締役会への助言、エグゼクティブコーチングにわたります。業界、地域、ビジネスモデルを超え、私の仕事は一貫して戦略的な野望を持続可能な成果に変えることに焦点を当てています。

私は30件以上の合併と買収のリードおよび助言を行い、企業が戦略的根拠の定義、厳密なデューデリジェンスの実施、取引の交渉、統合のリード、そして取引後のパフォーマンスの改善を通じて、フルライフサイクルを支援してきました。これらの経験は、複雑な取引のダイナミクスを実務的かつ実行可能にすることを目的とした私の著書、『合併と買収の神秘を解く』『中小企業のためのM&A』『M&A統合ハンドブック』の基盤を形成しています。
