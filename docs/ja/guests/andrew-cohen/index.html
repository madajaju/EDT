LinkedIn Profile

1997年以降、アンドリューはEVOKEのCEO兼マネージングパートナーを務めており、同社は受賞歴のあるデジタルトランスフォーメーション企業で、クライアントと提携してほぼすべての一般的な組織的および技術的な課題に対する近代的な職場ソリューションを構築しています。

2019年にアンドリューはEVOKE Engage部門を立ち上げ、プロセスの自動化と効率化に重点を置いています。彼は2022年初頭にネツリットに参加し、これによりネツリットはクライアントに技術を活用してROIを最大化するためのガイダンスとソリューションを提供できるようになりました。

アンドリューは以下の取締役会でも活動しています：赤十字社（ハドソンバレー）：理事会、People USA：理事会、Pathfinder FC：顧問委員会。

サッカーの熱心なファンでプレーヤーであるアンドリューは、オフィスにいないときはサッカーを観戦したりプレーしたりしているほか、家族と過ごす時間も大切にしています。サッカーや仕事の外では、油絵を描いたり、ゴルフを1ラウンドまたは2ラウンド楽しんだり、料理人を目指すことを楽しんでいます。
