Walter Hancock

Walter Hancock

LinkedIn Profile

1983年にコンピュータを発見し、その有用性を追求し続けています。過去30年間で、50社以上の大企業と中企業に対して株主価値を創出してきました。楽しい経験であり、その過程で多くのことを学びました。

高級言語の登場により、時間とお金さえあれば何でも可能であることが明らかになりました。私は、可能性の最前線にいることを楽しみ、それをシンプルにし、人々がその影響を理解できるように手助けすることが好きです。何かを行う異なる方法を追求することは、刺激的で困難であり、価値を加えるための最大の可能性を秘めています。

2009年にブロックチェーンを発見し、ここ数年で量子技術に出会いました。エンタープライズのベストプラクティスを念頭に置きながら、次の進化の可能性を思い描くと非常にワクワクします。私たちは、すべてを放棄することなく、人々が次のレベルに到達できるように支援するためにWalacor Corporationを設立しました。
