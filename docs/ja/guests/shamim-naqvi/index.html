Shamim Naqvi

Shamim Naqvi

LinkedIn Profile

経験豊富なCEO兼共同創業者であり、人工知能と機械学習の分野を推進し、その倫理的かつ責任ある利用を確保することに情熱を持っています。私は、安全なコンピューティングの力を活用した革新的なAIガバナンスプラットフォームであるSafeLiShareを率いることを誇りに思っています。

私たちの使命は、企業にとって、内側から外側へと、より安全で透明性のあるデジタルエコシステムを作り出すことです。強固な学術的背景とBell Labsでの15年以上のシニアリサーチ経験を持ち、最新技術とその実用的な応用に関する深い理解を提供します。物理学、電気工学、コンピュータサイエンスの学位を持ち、複雑な問題を解決するための高度なアルゴリズムとモデルの開発において専門知識を磨いてきました。

SafeLiShareでは、AIガバナンスの最前線に立ち、機密データを保護しながら、安全で協力的かつ共有されたAIワークフローを可能にするための機密コンピューティングの利用を先駆けています。プライバシー、透明性、説明責任の原則を組み合わせることで、組織がAIの可能性を活用しながら、自分たちのデータをコントロールし、規制に対する遵守を確保できるようにしています。

私は、公演やアドバイザリーの役割を通じて専門コミュニティに積極的に貢献しています。倫理的AIプラクティスにコミットする業界の専門家、政策立案者、組織と連携しながら、AIガバナンスの未来を形作ることに努めています。
