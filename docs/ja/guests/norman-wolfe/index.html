Biography

Fortune 500企業から急成長中のスタートアップまで、40年以上にわたり、私は一つの問いを探求してきました。なぜ、優れた戦略と有能なチームがあるにもかかわらず、望む成果を出すのに苦労するのでしょうか？

キャリアの初期にヒューレット・パッカードで働いていた頃、私は、実行が失敗する原因はたいてい努力や知性ではないことを発見しました。失敗の原因は、人それぞれが変化を異なる形で解釈することにあります。人はそれぞれの意味づけ、経験、前提を、あらゆる計画、あらゆる会話、あらゆる意思決定に持ち込むのです。

この洞察が、The Living Organization® Framework へとつながりました。このフレームワークは、あらゆる成果が次の3つの力によって形づくられることを示しています。 • Activity — 私たちが何をするか • Relationship — どのように関わるか • Context — 私たちの反応を駆動する意味と物語

簡単に言えば、成果は私たちの行動（Activity と Relationship）によって決まり、行動は Context によって動かされます。

長年にわたり、私は Quantum Leaders を通じて、CEOや経営チームと直接この取り組みを実践してきました。現在は、その焦点をさらに広げています。

私は、Living Organization® の実践者からなるグローバルコミュニティを構築しています。コーチ、コンサルタント、そして社内リーダーたちが、この取り組みを深く、誠実に、そして現実の成果につながる形で組織に持ち込めるよう育成しています。

経営チームと直接取り組む場合でも、実践者を育成する場合でも、私の仕事の中心にあるのは、リーダーが次のことを実現できるよう支援することです。 • パフォーマンスを形づくる隠れた文脈要因を明らかにする • 関係性のエネルギーを強化する • 実行力を高めるための能力と成熟度を構築する

もしあなたの戦略が、可能だと分かっている成果を生み出していないなら、問題は組織が何をしているかではなく、世界をどのように意味づけているかにあるのかもしれません。
