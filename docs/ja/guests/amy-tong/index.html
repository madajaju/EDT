Biography

エイミー・トンは2022年3月にガヴィン・ニューサム知事によって政府運営長官に任命され、州の最高情報責任者およびカリフォルニア技術省の長官から昇進しました。

彼女は1986年に中国からアメリカに移住した第一世代の移民で、8つのスーツケースだけを持ってやってきました。カリフォルニアに適応するために公共サービスに頼り、その経験が現在の公務員としてのコミットメントを促しています。

州の公務員としての彼女の20年以上にわたるキャリアは、カリフォルニア州の州水資源管理委員会でエントリーレベルのプログラマーとして始まりました。そこから、彼女は6つの州機関でより大きな責任を持つ役職に昇進し、最終的には行政のリーダーシップ役に任命されました。

長官になる前、トンはCOVID-19パンデミック中に、病院の収容能力の監視、公共健康報告、リモートラーニング、テレワーク、ワクチン接種、野火の対応に関するシステムを含む情報とデータを提供する重要な24時間体制の技術プロジェクトを指導しました。緊急時には、40以上の新しい技術ツールの構築とレガシーシステムの重要な修理を行う努力を導きました。

政府運営長官として、トンは州政府の革新を通じての変革に注力しています。

ユーザー中心のデザインの擁護者であるトン長官は、州のプロジェクト提供、技術調達、クラウドスマート採用、オープンデータ、労働力育成パイプライン、サイバーセキュリティ措置の近代化に向けた取り組みをリードしています。州政府での25年以上のキャリアを通じて、データに基づく意思決定の擁護者であり、カリフォルニアの住民のために新しくコスト効果の高い技術を活用し続けることを目指しています。

州政府でのリーダーシップ役に加え、トンはカリフォルニアノースステート大学の理事会、全米有色人種向上協会、アジア太平洋州雇用員協会、アジア太平洋島嶼米国民公務業務協会の全国シニアアドバイザーとしてボランティア活動を行っています。

トン長官は、サクラメントにあるカリフォルニア州立大学で経営情報システムの理学士号とビジネス管理の修士号を取得しています。
