Jared Shepard

Jared Shepard

LinkedIn Profile

ハイポリの社長兼CEOであるジャレッド・シェパードは、エッジからの安全なデータアクセスとインタラクションの変革を目指して、最先端の仮想モビリティSaaS企業を推進しています。シェパード氏は、20年以上の情報技術、連邦サービス、および営業の経験を持つ、経験豊富で革新的な業界リーダーです。彼はアメリカ陸軍の退役軍人であり、軍務を終えた後、アメリカ陸軍第3軍団のリードテクニカルプランナーになりました。ジャレッドは2003年から2005年の間、CJTF-7およびMNC-Iに派遣され、米国が構築した国防総省のイラクネットワークインフラを設計しました。2006年に、共通IED対策組織（JIEDDO）の重要なニーズを支援するためにインテリジェントウェーブを設立しました。彼は、最初のIED対策作戦情報センター（COIC）をMNC-Iに構築するために自ら派遣され、その後、イラクおよびアフガニスタンのすべての主要指揮官にCOICの能力を構築しました。ジャレッドは、戦闘員の戦術的および戦略的ニーズに基づいてインテリジェントウェーブを成長させました。彼は2021年にハイポリを設立しました。

ジャレッドは、退役軍人を中心とした501(c)(3)の非営利団体「ウォリアーズ・エトス」の設立者兼会長です。ジャレッドとハイポリは、2020年のアーンズト・アンド・ヤングの企業家年賞（ミッドアトランティック地域）、2021年のジョージ・C・マシャル国際センターのイノベーションリーダーシップ賞、2021年のファストカンパニーの最も革新的な企業セキュリティ部門での第6位、2022年の2つの金賞サイバーセキュリティエクセレンス賞、2016年から2020年までのモンスター.comおよびミリタリー.comによる「退役軍人向け全国トップ10企業」など、数々の賞を受賞しています。
