Biography

ラブ・カプールは2021年12月からBit（bit.dev）でソフトウェアエンジニアリングコンサルタント（R&D）として働いています。それ以前は、2017年から2021年12月までHOOPP（オンタリオ州年金プラン）で働いていました。HOOPPに在籍していた間、ラブはソリューションアーキテクト、シニアシステム開発者、プロダクトオーナー、副システム開発者、およびシステム開発者などさまざまな役割を担っていました。ラブは、投資の意思決定、運用効率の自動化と改善、運用リスクの低減を目的としたアプリケーションとサービスを構築する投資技術チームの一員でした。彼はフルスタック開発を担当し、アジャイル環境でのカスタムフロントオフィスアプリケーションの設計と開発を行っていました。ラブの主要な成果には、流動性管理と取引プラットフォームのフロントエンドアーキテクチャとデザインのリーダーシップ、分析とコンプライアンスチェックを追加することによる社内カスタム株式取引アプリケーションの維持と強化、パフォーマンスとコンプライアンス検証の最適化によるユーザーエクスペリエンスの改善が含まれます。

ラブ・カプールの教育歴は、2011年にインドのニューデリーにあるセント・フランシス・デ・セールズ・スクールに入学したことから始まります。ラブは2011年から2013年の2年間そこで過ごし、コンピュータサイエンスに焦点を当てて高校教育を修了しました。

高校を卒業した後、ラブは2013年から2017年までセネカカレッジでさらに教育を追求しました。この4年間で、ソフトウェア開発の分野で名誉学士号（B.Tech.）を取得しました。

提供された情報に基づくと、ラブ・カプールの教育歴には、セント・フランシス・デ・セールズ・スクールでコンピュータサイエンスに焦点を当てた高校教育を修了し、セネカカレッジでソフトウェア開発の学士号を取得したことが含まれています。
