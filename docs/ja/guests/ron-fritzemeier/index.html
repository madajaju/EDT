Ron Fritzemeier

LinkedIn Profile

ロンは2022年にインテルに入社し、インテルの公共部門防衛および国家安全保障グループの一員です。彼の役割は、デジタルトランスフォーメーションを支援し、助言することで、データの安全で戦略的な管理、分析、および活用を追求し、重要なミッションの課題を解決するために国防総省および情報コミュニティのリーダーをサポートすることです。

カリフォルニア州カミノ出身のロンは、カリフォルニア州立大学フレズノ校で電気工学の理学士号を取得し、その後スタンフォード大学で電気工学の修士号を取得しました。ロンは1983年にサンディア国立研究所に入社し、放射線耐性マイクロエレクトロニクスのテスト、設計、テスト可能性の設計に従事しました。彼は後にラボのフィールドインテリジェンスエレメントに参加し、ハードウェアおよびソフトウェアのリスク分析とリバースエンジニアリングを専門としました。1994年にロンは情報コミュニティに移行し、政府の社内コンピュータエンジニアリングラボを設立し、情報収集作戦のためのツールと技術を作成しました。シニアインテリジェンスサービスの役員として、ロンは世界的な情報操作ユニットを指導しました。その後、彼は同機関のサイバー戦略の責任者を務め、ブッシュ大統領とオバマ大統領の両政権における大統領のサイバー interagency ポリシー委員会で同機関を代表しました。

並行して、ロンは海軍予備役の情報戦専門の技術士官でした。約36年間にわたる海軍のサービスの中で、ロンは艦隊の保守および調達、C4Iおよび武器システム、サイバー作戦およびサイバーセキュリティを支援するいくつかの指揮職を務めました。2016年、ロンは将官に昇進し、現役に復帰しました。この職務において、彼は現在の海軍情報戦システム指令の主任技術者として服務し、その後、米国戦略司令官の下で核司令、制御および通信企業センターのディレクターを2年間務めました。ロンは2021年秋に海軍を退役しました。
