Jordan Burris

Jordan Burris

LinkedIn Profile

ジョーダン・バリスは、政府のイノベーションとデジタルトラストにおいて先駆者であり、15年以上の経験を活かして公共部門のテクノロジーを変革し、最先端のソリューションを開発しています。ソキュアの公共部門の副社長兼ゼネラルマネージャーとして、ジョーダンはアイデンティティの包括性と公正性に深くコミットしています。この役割では、ビジネスの運営と戦略的方向性を導きながら、戦略、顧客の成功、営業、ビジネス開発、法務、コンプライアンスのダイナミックなチームをシームレスに統合し、アイデンティティ検証技術がデジタルサービスの提供をどのように再定義するかを模索しています。

ソキュアに入る前は、ホワイトハウスで米国最高情報責任者（CIO）のチーフ・オブ・スタッフを務め、2つの大統領政権にわたって変革的なテクノロジーとサイバーセキュリティの取り組みを主導しました。連邦ITインフラの近代化、サイバーセキュリティの再構築、デジタルアイデンティティソリューションの推進などのイニシアティブの重要な設計者であり、連邦政府の数十億ドル規模のテクノロジー予算を監督していました。

テクノロジーの著名な声として、ジョーダンはアイデンティティ盗難リソースセンターとカンターライニシアティブの取締役を務め、アイデンティティ侵害の軽減とデジタルアイデンティティ基準の推進に積極的に取り組んでいます。CISSPおよびCRISCの認定プラクティショナーであり、ペンシルベニア州立大学の誇り高い卒業生であるジョーダンは、安全でより包括的なデジタル未来を形作る使命に取り組んでいます。
