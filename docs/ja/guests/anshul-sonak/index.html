Anshul Sonak

Biography

私は、デジタルレディネス、人工知能、人間と機械のパートナーシップ、未来の仕事に関する学習とスキル、および社会的影響に関する専門知識を持つ、グローバルなテクノロジースキルビジネスリーダーおよびプログラムデザイナーです。シリコンバレーのインテル本社に基づき、デジタルレディネスプログラムおよび未来の仕事に関する初のプリンシパルエンジニアを務めています。政府とのパートナーシップを持つグローバルなAIスキリングアジェンダを管轄しています。私は、世界中の産業、学術、政府向けにAIスキリングモデルを開発しました。私は公共と民間のパートナーシップを創出し、ビジネス開発および市場拡大の革新者であり、思想的リーダーシップや企業の評判を向上させる能力を持っており、労働政策開発に関する政府関係の専門家でもあります。以下の分野で優れた成果を上げています：

ビジネス変革：私は、公共-民間パートナーシッププログラムの設計における成果志向のリーダーシップ経験を持ち、ビジネスと評判の向上を支援し、ポリシーの提唱や新しい包括的イノベーションモデルをビジネス、技術革新、開発アジェンダを統合することで実現しています。

思想的リーダーおよび尊敬される講演者：私は技術革新とスキル-仕事の変革、若者のエンパワーメント、教育の変革、不平等の削減に情熱を注いでいます。政府、市民社会、学界にとって新しい変革モデルを構築する信頼できるアドバイザーとして認識されています。私は政府がデジタルレディネスと労働政策、責任あるAI技術エコシステム、競争優位のための新しい学習スキリング環境を発展させるのを支援します。業界、学術、政府、国際フォーラム（UNDP、UNESCO、UN ESCAP、ADB、USAIDなど）で基調講演を行っています。私は国連開発計画（UNDP）アジアの初のユースコラボチャンピオンです。また、MIT SOLVEのグローバルジャッジでもあります。私は、Forbes、Wharton、国連などのさまざまなグローバルな出版物やプラットフォームで執筆しています。

共有価値を通じたステークホルダー管理：私は優れたアドボカシー、コミュニケーション、チームビルディングスキルを持っており、さまざまなレベルや機能、地理、文化を超えて強固な関係を構築し、決定に影響を与え、組織を構築することができます。私は多くの社会的影響を与える組織やビジネスリーダーにとって尊敬されるコーチおよびメンターです。

27年間のグローバルなキャリアとリーダーシップの旅：私はインド、アメリカ、ドバイ（UAE）、クアラルンプール（マレーシア）、シンガポールに住んでいました。世界50か国以上を旅してグローバルリーダーと会っています。私は、教育、スキルテクノロジー、AI、および社会的影響セクターにおける選ばれた社会的影響のある組織に対するボードアドバイザーでもあります。
