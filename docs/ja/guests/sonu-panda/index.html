ソヌ・パンダは、エネルギーを節約し、炭素排出を削減し、居住者の健康や快適さを犠牲にすることなくコストを削減しながら、ポートフォリオがESG目標を達成できるように支援する、建物の運用パフォーマンスを最適化する受動データのCEOです。彼は受動データのビジネスと技術戦略を策定・実行し、営業、ビジネス開発、マーケティング、顧客のオンボーディングと成功、エンジニアリング、タレントマネジメント、財務、行政業務など、会社内のすべての機能を監督しています。受動データに先立ち、パンダは最近、B2Bマーケットプレイス企業であるH.Bloomの共同創設者兼COOを務め、同社の運用モデルを開発し、製品/技術開発およびプロセスエンジニアリングの取り組みを確立し、日常のビジネス運営を管理しました。
